Description :- This is a simple and responsive portfolio webpage built using HTML, plain CSS, and vanilla JavaScript without the use of any frameworks or libraries. The project focuses on clean design, structured layout, and smooth user interaction while keeping the code lightweight and easy to understand.

The webpage includes essential sections such as a navigation bar, about section, skills, and contact information. Styling is done entirely with CSS, ensuring responsiveness and visual consistency across different screen sizes. JavaScript is used to add basic interactivity and dynamic behavior where needed.

This project demonstrates a strong understanding of core web development fundamentals, including semantic HTML structure, modern CSS styling techniques, and pure JavaScript functionality — all implemented without relying on external frameworks.

Features Implemented:

Introduction Section – A brief personal introduction highlighting my background and career goals.

Skills Section – Showcases my technical skills in a clear and organized manner.

Education Section – Displays my academic background and qualifications.

Experience Section – Highlights my professional or project experience.

Responsive Navigation Bar – A scalable and user-friendly navbar that ensures smooth navigation across different screen sizes.

Contact Form – Allows visitors to directly connect with me through the website.

Social Media Integration – Quick access links to my Instagram, GitHub, and LinkedIn profiles.
